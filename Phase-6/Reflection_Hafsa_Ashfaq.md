
# Individual Reflection - Hafsa Ashfaq

## Technical Skill I Improved Most

During this project, I improved my understanding of how a trained deep learning model can be converted into a usable web application. I learned how frontend design, backend routing, preprocessing, model inference, and report generation connect together.

## Most Important Decision

The most important decision was selecting ResNet-18 full fine-tuning as the final model because it gave the best validation performance and was lightweight enough for deployment.

## Where I Got Stuck

The longest difficulty was deployment and UI integration. Issues such as template errors, internal server errors, report generation, and dark mode visibility required debugging and repeated testing.

## What I Would Do Differently

If starting again, I would plan the repository structure earlier and keep every phase organized from the beginning.

## What Surprised Me

I learned that deep learning projects are not only about model accuracy. Deployment, preprocessing consistency, user interface, error handling, and documentation are equally important.
