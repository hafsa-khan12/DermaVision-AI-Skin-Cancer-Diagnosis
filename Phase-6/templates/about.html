AI Medical Screening Project

DermaVision AI

Smart Skin Cancer Diagnosis & Risk Screening System

DermaVision AI is a deep learning powered web application designed to support early skin cancer screening. It analyzes skin lesion images, predicts benign or malignant class, calculates confidence score, explains prediction basis, and generates a professional medical-style report.

Project Objective

To build a professional AI-based diagnosis interface that supports early screening of skin cancer using image classification, confidence scoring, and risk assessment.

⚕️

Problem Statement

Skin cancer symptoms can visually overlap with harmless skin lesions. This system demonstrates how deep learning can assist in medical image screening workflows.

🧠

AI Approach

The system uses a ResNet-18 transfer learning model trained to classify skin lesion images into benign and malignant categories.

System Modules

How the Application Works

Complete diagnosis workflow from image upload to report generation.

📤

Image Upload

Upload JPG, PNG, or JPEG skin lesion images.

⚙️

Preprocessing

Resize, normalize, and prepare image for AI inference.

🧠

AI Prediction

ResNet-18 predicts benign or malignant lesion class.

📊

Confidence Score

Probability-based confidence score is generated.

⚠️

Risk Assessment

Risk level is assigned based on prediction confidence.

📄

Report Generation

Printable professional medical-style report is generated.

Technology Stack

Tools & Frameworks

FastAPI PyTorch Torchvision ResNet-18 HTML CSS JavaScript PIL

The frontend is designed using custom HTML/CSS, while FastAPI handles backend routing and PyTorch manages model inference.

Methodology

Development Flow

01

Dataset organized into benign and malignant classes.

02

Images resized to 224 × 224 and normalized.

03

Transfer learning applied using pretrained ResNet-18.

04

Model integrated with FastAPI for real-time prediction.

05

Dashboard, analytics, report, and scan pages designed.

Project Impact

Why This Project Matters

This project demonstrates how AI can support healthcare screening workflows by providing fast image-based prediction, risk-level explanation, and structured reporting. It is useful for academic learning, prototype demonstration, and understanding AI-assisted medical decision support systems.
