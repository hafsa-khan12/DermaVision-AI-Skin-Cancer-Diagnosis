Conclusion

The final ResNet-18 full fine-tuning configuration achieved the strongest validation and final testing performance. Compared with the baseline model, the final model improved accuracy by 21.82%, showing that transfer learning significantly improved classification capability for skin lesion diagnosis. The model is suitable for educational screening support, but it should not be used as a replacement for clinical diagnosis.
