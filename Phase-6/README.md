## Final Application

DermaVision AI is a skin cancer diagnosis and risk screening web application. It allows users to upload a skin lesion image and receive:

- Predicted class
- Confidence score
- Risk level
- Class probabilities
- Preprocessing pipeline preview
- AI-generated medical summary
- Printable medical-style report

## Final Model

The selected final model is ResNet-18 Full Fine-Tuning.

## Key Results

- Final Accuracy: 89.7%
- Macro F1-score: 89.59%
- Baseline Accuracy: 67.88%
- Accuracy Improvement: +21.82%
- Single Image Latency: 0.070 seconds
- Checkpoint Size: 42.97 MB
