
# Phase 2 - Data, EDA and Baseline Model

This folder contains Phase 2 work for DermaVision AI.

## Phase 2 Focus

- Dataset acquisition and documentation
- Exploratory Data Analysis
- Preprocessing pipeline
- Baseline MLP model
- Training logs and evaluation
- Progress report

## Contents

- `DATA_CARD.md` contains dataset description.
- `notebooks/` contains EDA notebook and HTML export.
- `src/` contains dataset loading, preprocessing, baseline model, training, and seed utilities.
- `configs/` contains baseline model configuration.
- `reports/` contains figures and results.
- `experiments/` contains training logs.
- `tests/` contains preprocessing smoke test.

This project is for academic and educational purposes only.
