# DermaVision AI

DermaVision AI is a deep learning-based skin cancer diagnosis and risk screening system.

## Project Goal

The goal of this project is to design, train, evaluate, and deploy a deep learning system that can classify skin lesion images as benign or malignant.

## Phase 1

This repository currently contains the Phase 1 roadmap structure, documentation folders, planned source code structure, configuration folder, reports folder, and testing folder.

## Planned System

- Baseline MLP model
- CNN model
- Transfer learning using ResNet-18
- Autoencoder / VAE component
- Evaluation and error analysis
- Final deployment using FastAPI web application

## Repository Structure

- `docs/` contains roadmap and planning documents.
- `data/` contains raw and processed dataset folders.
- `notebooks/` contains experiment notebooks.
- `src/` contains source code modules.
- `configs/` contains configuration files.
- `reports/` contains report files and figures.
- `tests/` contains test files.
