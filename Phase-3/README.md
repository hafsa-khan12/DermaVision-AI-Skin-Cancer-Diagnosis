# Phase 3 - Deep Architecture and Model Training

This folder contains Phase 3 work for DermaVision AI.

## Phase 3 Focus

- Implement custom CNN architecture
- Implement transfer learning using ResNet-18
- Train deep learning models on skin lesion images
- Compare CNN and ResNet-18 performance
- Generate training curves, accuracy curves, and confusion matrix
- Document architecture decisions and results

## Contents

- `notebooks/` contains Phase 3 training notebook and HTML export.
- `src/` contains CNN, ResNet, training, evaluation, and seed utility scripts.
- `configs/` contains model configuration files.
- `reports/` contains architecture diagrams, curves, and evaluation figures.
- `experiments/` contains training logs.
- `tests/` contains model smoke tests.

## Models Used

1. Custom CNN
2. ResNet-18 Transfer Learning
